# 🌿 NSBM Green University — Website

> **South Asia's first green university, now on the web.**  
> A fully responsive, multi-page university website built from scratch using pure HTML & CSS — featuring a working login system and student dashboard.

---

## 🚀 Live Pages

| Page | File | Description |
|------|------|-------------|
| 🏠 Home | `Web_assignment.html` | Hero section, stats strip, news, quick links |
| 📖 About Us | `about_us.html` | University story, values, campus stats |
| 🎓 Courses | `courses.html` | Foundation, Undergraduate & Postgraduate |
| 🏛️ Facilities | `facilities.html` | Lecture halls, sports, library & more |
| 👩‍🏫 Faculties | `faculties.html` | Computing, Business, Engineering, Science |
| 📞 Contact Us | `contact_us.html` | Contact form + Google Maps embed |
| 🔐 Login | `login.html` | Sign in, Register & Forgot Password |
| 📊 Dashboard | `dashboard.html` | Student portal after login |

---

## ✨ Features

- ✅ **Consistent white navbar** with green underline hover & active highlight across all pages
- ✅ **Identical footer** on every page — logo, quick links, contact info, social icons
- ✅ **Hero sections** with zoom animation on every page
- ✅ **Login system** — Sign In, Register (with password strength meter), Forgot Password
- ✅ **Student Dashboard** — timetable, announcements, module results, attendance bars, GPA stats
- ✅ **Contact form** with live success alert
- ✅ **Responsive design** — works on desktop, tablet & mobile
- ✅ **Smooth animations** — card hovers, nav transitions, hero zoom

## ⚡ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/nsbm-green-university-website.git
   ```

2. **Open the project folder**
   ```bash
   cd nsbm-green-university-website
   ```

3. **Launch in your browser**
   ```bash
   # Simply open Web_assignment.html in any browser
   open Web_assignment.html
   ```
   > No build tools, no npm, no setup required. Just open and go. 🎉

---

## 📌 Notes

- All images (logos, backgrounds, facility photos) must be placed in the **same folder** as the HTML files.
- The login system uses **sessionStorage** — credentials reset when the browser tab is closed.
- The Google Maps embed on the Contact page requires an internet connection to load.
